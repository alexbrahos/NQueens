To run the program, run "make run" or execute runMe and type in an integer when prompted. This will be both the dimensions of the board and the number of queens.

The result will be a representation of a chess board where 0 is an empty space and 1 is the location of a queen. The dimensions of the board will match the number of queens. The position of the queens will be spread out so that no queen can directly attack another queen.
